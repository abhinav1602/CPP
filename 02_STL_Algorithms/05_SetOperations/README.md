# ðŸ”— Set Operations

Algorithms for sorted ranges (set operations).

## Algorithms Covered

- `std::set_union` - Union of two sorted ranges
- `std::set_intersection` - Intersection of two sorted ranges
- `std::set_difference` - Elements in first but not second
- `std::set_symmetric_difference` - Elements in one but not both

**Note:** Input ranges must be sorted!

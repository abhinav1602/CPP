Algorithms for ordering elements in a range.

## Algorithms Covered

- `std::sort` - Sort range (O(n log n))
- `std::stable_sort` - Sort preserving order of equal elements
- `std::partial_sort` - Sort first k elements
- `std::nth_element` - Place nth element in sorted position
- `std::partition` - Partition by predicate
- `std::binary_search` - Search in sorted range (O(log n))

## Complexity

- `sort`: O(n log n)
- `partial_sort`: O(n log k)
- `nth_element`: O(n) average
- `binary_search`: O(log n)

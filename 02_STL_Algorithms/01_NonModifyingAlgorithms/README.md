# üîç Non-Modifying Algorithms

Algorithms that inspect elements but do not modify them.

## Algorithms Covered

- `std::find` - Find first element equal to value
- `std::find_if` - Find first element satisfying predicate
- `std::count` - Count occurrences of value
- `std::count_if` - Count elements satisfying predicate
- `std::search` - Find subsequence in range
- `std::all_of`, `std::any_of`, `std::none_of` - Check predicates

## Complexity

All algorithms: **O(n)** where n is the size of the range.
